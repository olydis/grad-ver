type aliasprop_set = AliasPropSet.t 
type aliasing_context = {
   parent : aliasing_context option;
   scope_id : Ast.scope_id;
   props : aliasprop_set;
   children : (aliasing_context_label * aliasing_context) list;
}

An aliasing context A is a set of aliasing propositions and a set of labeled child contexts. This forms a tree structure.

type aliasing_context_label =
| ACL_Condition of Ast.expression
| ACL_Unfolding of Ast.predicate_check

Each branch off of a parent aliasing context is labeled by either the condition or the "unfolding ... in ..." formula needed to go down the branch.

val propsEntailsAliased : aliasprop_set -> aliasprop -> bool

Judge whether a set ps of aliased propositions entail a given aliased proposition p. This is calculated by finding the existence or non-existence of a member p' of ps such that p is a subset of p'.

val entailsAliasProp : aliasing_context -> aliasprop -> bool

Evaluates the judgement that a given aliasing context entails that an aliasing proposition is true. In other words, finds an element (object variable set) of the total aliasing proposition set of the context that is a superset of the given aliasing proposition.

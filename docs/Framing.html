Permissions

type permission =
| Accessed of Ast.expression_field_reference
| Assumed of Ast.predicate_check

A permission is either to access a field to assume a predicate. Note that the permission to assume a predicate is derived permission - it expands into the permissions granted by the singly-unrolled body of the predicate.

Framing

val frames : Wellformed.ClassContext.t ->
       Wellformed.TypeContext.t ->
       Aliasing.ScopingContext.t ->
       Aliasing.AliasingContext.t -> PermissionSet.t -> Ast.formula -> bool

frames perms phi = true if and only if perms frames phi i.e. each permission required to frame phi is an element of either perms or granted phi

val selfFrames : Wellformed.ClassContext.t -> Wellformed.TypeContext.t -> Ast.formula -> bool

selfFrames phi = true if and only if phi grants all the permissions it requires i.e. the empty set frames phi.

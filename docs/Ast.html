Scoping withing Formulas

Scopes are only tracked within formulas for the purpose of reasoning about aliasing. The root level of a formula has the root_scope. Nestings within a formula each have unique (within the formula) scopes.

type scope =
| Scope of int 
type 'a enscoped = 'a * scope 
include ??
val string_of_scope : scope -> string
val scopeOf : 'a enscoped -> scope
val termOf : 'a enscoped -> 'a
module ScopeGenerator: sig .. end

A ScopeGenerator is used to create the ids used to identify the scopes of nested formulas.
